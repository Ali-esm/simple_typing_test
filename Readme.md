
# Typing Test

<p>
simple typing test app in terminal using python.
</p>

# How to run

* windows users need to install curses with pip

    ``` pip install curses```
  
1. clone the project

   ```git clone https://github.com/Ali-esm/simple_typing_test.git```

2. run in terminal

    ```python simple_typing_test.py```
